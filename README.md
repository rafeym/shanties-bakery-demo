# [Shantie's Bakery Demo](https://shanties-bakery-demo.vercel.app/)

This repo is a demo version of a deployed project that I have worked on for a small bakery business called Shantie's Bakery.

## Features ⭐

- Guest Checkout
- Shopping Cart
- Admin Dashboard
- Order fulfillment/cancellation
- Email Integration
  - Order status emails
  - Newsletter emails
- CRUD Functionality
  - Inventory
  - Orders
- Mobile Compatibile

## Tech Stack 👨‍💻

### Client

- React.js
- SASS
- Deployed on Vercel

### Server

- Node.js
- Express.js
- MongoDB
- Deployed on Heroku
